## ✨ معرفی

**Bootstrap Quest** یک بازی تعاملی تحت وب برای یادگیری **Bootstrap 5** است که شامل **۲۵ مرحله** و بیش از **۱۷۵ چالش** می‌باشد. 🎯

این بازی بر اساس آموزش‌های [W3Schools Bootstrap 5](https://www.w3schools.com/bootstrap5/index.php) طراحی شده و تمام مفاهیم از Grid System تا کامپوننت‌های پیشرفته مانند Modal، Carousel و Offcanvas را پوشش می‌دهد.

### 🎓 مسیر یادگیری

- 🚀 **مبانی** - شروع Bootstrap 5، Container و Breakpoints
- 🔲 **چیدمان** - Grid System پایه و پیشرفته (۱۲ ستونه)
- 🎨 **قالب‌بندی** - Typography، Colors و Backgrounds
- 📦 **کامپوننت‌ها** - Tables، Images، Alerts، Buttons، Badges
- 📊 **پیشرفته** - Pagination، List Groups، Cards، Dropdowns
- 🧭 **ناوبری** - Navs، Navbar واکنش‌گرا
- 📋 **فرم‌ها** - Form Controls، Input Groups، Validation
- 🎠 **تعاملی** - Carousel، Modal، Offcanvas، Scrollspy
- 🛠️ **ابزارها** - Spacing، Flexbox، Display Utilities
- ✨ **آیکون‌ها** - Bootstrap Icons و Utilities

## ✨ ویژگی‌ها

- 📚 **۲۵ مرحله جامع** - از مبتدی تا پیشرفته
- 🧩 **۱۷۵+ چالش** - ترکیبی از کوئیز و تمرین کدنویسی
- 💾 **ذخیره خودکار پیشرفت** - با استفاده از localStorage
- 🏆 **سیستم گیمیفیکیشن** - ستاره، XP، streak و نشان‌های دستاورد
- 📱 **کاملاً ریسپانسیو** - سازگار با موبایل، تبلت و دسکتاپ
- ⚡ **بدون نیاز به اینترنت** - فایل HTML واحد
- 🎨 **رابط کاربری مدرن** - تم تیره با رنگ‌های Bootstrap
- 🚫 **بدون وابستگی** - HTML/CSS/JS خالص، بدون فریم‌ورک

## 🚀 اجرا

فایل `index.html` را دانلود و در مرورگر باز کنید. همین! ✅

> **نکته:** برای جلوگیری از هشدار امنیتی مرورگر، می‌توانید با Live Server یا `python -m http.server` اجرا کنید.
